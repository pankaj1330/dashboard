## ON Local Machine 
## Steps

1. **Make sure you have node installed on your machine**

2. **Install Dependencies**
   ```
   npm install
   ```

3. **Run Development Server**
   ```
   npm run dev
   ```
   Visit `http://localhost:5173` in your browser.


## Technologies Used

- React.js
- React Router
- CSS Modules / Styled Components
